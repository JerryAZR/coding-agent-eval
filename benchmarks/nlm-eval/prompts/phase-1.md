# Phase 1: Simple Binary Operations

Build a CLI tool named `nlm-eval`.

It must support:
- `./nlm-eval NUM1 plus NUM2`
- `./nlm-eval NUM1 minus NUM2`

`NUM1` and `NUM2` are real numbers (integers or decimals like `1.5`).

Print the numeric result to stdout and exit 0 on success.
Exit non-zero on invalid input.
